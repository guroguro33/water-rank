# 全国水道料金ランキング
 
## アプリケーションの概要
 
vue.jsを使用した居住地域を選択すると水道料金の全国順位が表示されるサイトです。
[デモページ](https://kurosuke-web.com/water-rank/)

## 画面イメージ ##
 
![TOPイメージ](https://user-images.githubusercontent.com/48667277/87269768-22f3c500-c509-11ea-9999-c40f9599be46.png)
 
## 機能と環境
 
- SPA（vue.jsおよびvue router）
- vueアニメーション（vue.js）
- データ取得（axios）
- データベース（MySQL）
- 開発環境: Vue CLI
- データベース: MySQL
- デモページ環境: Xsever
 
## 必要要件
 
- vue: 2.6.11
- vue-axios: 2.1.5
- vue-router: 3.1.6
- MySQL: 5.7.26
- Apache: 2.2.34
- axios: 0.19.2
- babel-eslint: 10.1.0
- eslint: 6.7.2
- node-sass: 4.12.0
 
## 使い方
 
1. トップページから「はじめる」をクリック
2. 都道府県を選択
3. 市町村等を選択
4. 結果が表示！
